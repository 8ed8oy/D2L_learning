# 01 模型设计 - 层和块、自定义层

## 概要

PyTorch用块（Block）组织模型。块可以是单层、层的组合或整个模型。

---

## 关键概念与代码

### 1) Module 基类

```python
class MyLayer(nn.Module):
    def __init__(self):
        super().__init__()

    def forward(self, X):
        return X
```

要点：`__init__()`注册子模块与参数；`forward()`定义前向；反向由自动微分完成。

### 2) Sequential 容器

```python
net = nn.Sequential(
    nn.Linear(20, 256),
    nn.ReLU(),
    nn.Linear(256, 10)
)
out = net(X)
```

要点：按顺序连接层，适合线性拓扑模型。

### 3) 无参数层

```python
class CenteredLayer(nn.Module):
    def __init__(self):
        super().__init__()

    def forward(self, X):
        return X - X.mean()
```

要点：不含可训练参数，常用于数据变换或简单计算。

### 4) 有参数层

```python
class MyLinear(nn.Module):
    def __init__(self, in_units, out_units):
        super().__init__()
        self.weight = nn.Parameter(torch.randn(in_units, out_units))
        self.bias = nn.Parameter(torch.randn(out_units))

    def forward(self, X):
        return F.relu(X @ self.weight + self.bias)
```

要点：用 `nn.Parameter` 注册可训练参数，自动加入优化器。

### 5) 自定义块组合

```python
class MyBlock(nn.Module):
    def __init__(self):
        super().__init__()
        self.net = nn.Sequential(nn.Linear(10, 20), nn.ReLU())
        self.out = nn.Linear(20, 5)

    def forward(self, X):
        return self.out(self.net(X))
```

---

## 小结

- `nn.Module` 是所有模型与层的基类。
- `nn.Sequential` 适合顺序结构。
- 自定义层与块只需实现 `forward()`。
- 用 `nn.Parameter` 注册可训练参数。
